# CoolDrinks AI Website 🥤

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen.svg)](https://jayavarshini-k-s.github.io/CoolDrinks_AI_website/)
[![Next.js](https://img.shields.io/badge/Next.js-15.5-black?logo=next.js)](https://nextjs.org/)
[![Firebase](https://img.shields.io/badge/Firebase-Supported-FFCA28?logo=firebase&logoColor=black)](https://firebase.google.com/)

A modern, interactive parallax website showcasing a functional soda brand inspired by classic flavors. Built with a focus on stunning aesthetics, smooth animations, and high performance.

## 🚀 Live Demo

Check out the live project here: **[CoolDrinks AI Website](https://jayavarshini-k-s.github.io/CoolDrinks_AI_website/)**

## ✨ Features

- **Immersive Parallax Scrolling**: Dynamic background and foreground elements that react to user scroll.
- **Modern UI/UX**: Built with a sleek, dark-mode-first approach using Tailwind CSS.
- **Fully Responsive**: Optimized for seamless viewing on desktops, tablets, and mobile devices.
- **Static Export**: Hosted lightning-fast on GitHub Pages using automated GitHub Actions.

## 🛠️ Technology Stack

This project was brought to life using a modern and powerful set of tools:

- **Framework**: [Next.js](https://nextjs.org/) (React)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Backend & Integrations**: **Firebase** 
- **Workflow & Design**: Accelerated and crafted using **Flow**, **Whisk**, and **Nanobanana** to design and deliver beautiful user experiences.

## 📦 Local Development

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Jayavarshini-k-s/CoolDrinks_AI_website.git
   cd CoolDrinks_AI_website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open [http://localhost:9002](http://localhost:9002) in your browser.

## 🚀 Deployment

This project is automatically deployed to **GitHub Pages** using a custom GitHub Actions workflow (`.github/workflows/nextjs.yml`). Any push to the `main` branch will automatically build and publish the latest version of the site, correctly handling asset paths for GitHub Pages.
